    Bash - Mount Disk and Drives

    List disks

    sudo fdisk -l

    Create folder

    sudo mkdir /tmpmount

    Mount

    sudo mount /dev/sdb /tmpmount

    Unmount

    umount /dev/sdb

    Alternatively:

    umount /tmpmount
